# ☀️ Kindastuff Solar Analytics

**Free, private, browser-based calculators for solar PV professionals.**

No sign-ups. No data collection. All calculations run locally in your browser.

🌐 **[kindastuff.com](https://kindastuff.com/)**

---

## 🔧 Available Calculators

| Tool | What It Calculates |
| :-- | :-- |
| [CUF / PLF](https://kindastuff.com/conventional-cuf-or-plf/) | Standard Capacity Utilization Factor |
| [Degradation-Corrected CUF](https://kindastuff.com/degradation-insolation-corrected-cuf/) | CUF adjusted for module aging and insolation variance |
| [Specific Yield & PTI](https://kindastuff.com/yield-performance-metrics/) | kWh/kWp and Performance Target Index |
| [Capacity-Based PR](https://kindastuff.com/capacity-based-pr/) | Performance Ratio using DC capacity and POA |
| [Efficiency-Based PR](https://kindastuff.com/efficiency-based-pr/) | PR using module area and efficiency |
| [Instantaneous PR](https://kindastuff.com/instantaneous-pr/) | Real-time performance monitoring |
| [Temperature-Corrected PR](https://kindastuff.com/temperature-corrected-pr/) | PR normalized for cell temperature |
| [Soiling Analysis](https://kindastuff.com/soiling-analysis/) | Energy loss from dust and debris accumulation |
| [Plant & Grid Generation Loss](https://kindastuff.com/plant-grid-generation-loss/) | Loss attribution between plant-side and grid-side |
| [Plant & Grid Availability](https://kindastuff.com/plant-and-grid-availability/) | Downtime analysis against available irradiance |
| [Solar Insolation (POA)](https://kindastuff.com/solar-insolation-calculator/) | Reverse-calculate POA from plant output |
| [NMGG Calculator](https://kindastuff.com/nmgg-calculator/) | Monthly Net Minimum Guaranteed Generation targets |

## 🔒 Privacy-First Architecture

Unlike most solar software that requires account creation and data uploads, **Kindastuff runs entirely in your browser.**

- ✅ No server-side processing
- ✅ No database storage
- ✅ Plant data never leaves your device
- ✅ Standard Google Analytics for basic traffic measurement only
